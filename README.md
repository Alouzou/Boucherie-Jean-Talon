# BoucherieJeanTalon
### Présentation du projet
Ce projet est un site vitrine développé en Angular pour la Boucherie Jean Talon. Il met en avant les produits artisanaux de la boucherie — viandes halal, volailles, fruits, légumes et épicerie maghrébine — dans un design responsive et moderne. Le site reflète l'identité de la maison Chez Koceila, avec une navigation fluide et une présentation claire des spécialités proposées.
